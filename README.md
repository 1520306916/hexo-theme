#快速入门
- 需要在站点的 `_config.yml` 中添加：
```
archive_generator:
  per_page: 0
```
- 在主题的 `_config.yml` 中配置导航链接：
```
menu:
  首页: /
  标签: /tags
  归档: /archives
```
